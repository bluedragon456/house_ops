# HouseOps

HouseOps is a Home Assistant custom integration for tracking household equipment maintenance, creating actionable entities, and generating a purpose-built dashboard for operations and upkeep.

## Included in V1

- Furnace / HVAC tracking
- Water heater tracking
- Fire alarm / smoke detector tracking
- Config flow and options flow asset management
- Asset and task entities for dashboards, automations, notifications, and To-do style workflows
- HACS-ready packaging
- GitHub validate and release workflows
- Dynamic Lovelace strategy at `www/house_ops/house-ops-strategy.js`
